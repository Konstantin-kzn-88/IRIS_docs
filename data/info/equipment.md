Файл equipment.json предназначен для описания технологического оборудования опасного производственного объекта, содержащего опасные вещества, и используется для:

моделирования аварийных сценариев;

расчётов проливов, испарения и выбросов;

пространственного анализа размещения оборудования;

связи с базой данных веществ (substance_id).

Общая структура

Файл представляет собой массив объектов оборудования:

[
  { /* объект оборудования */ },
  { /* объект оборудования */ }
]


Каждый объект массива описывает одну единицу оборудования.

Структура объекта оборудования
1. Идентификация
Поле	Тип	Описание
id	integer	Уникальный идентификатор оборудования
substance_id	integer	Идентификатор вещества из справочника веществ
equipment_name	string	Условное/технологическое наименование оборудования
2. Фазовое состояние
Поле	Тип	Описание
phase_state	string	Фазовое состояние вещества:
• ж.ф. — жидкая фаза
• г.ф. — газовая фаза
• ж.ф.+г.ф. — двухфазное состояние
3. Тип задания координат
Поле	Тип	Описание
coord_type	integer	Тип геометрии объекта

Допустимые значения coord_type:

Код	Описание
0	Линейный объект (начало–конец)
1	Точечный объект
2	Стационарный объект (площадь, первая и последняя координаты замкнуты)
4. Координаты размещения
Поле	Тип	Описание
coordinates	array[number]	Координаты размещения оборудования

Интерпретация массива coordinates:

coord_type = 0 → [x1, y1, x2, y2]

coord_type = 1 → [x, y]

coord_type = 2 → [x1, y1, x2, y2] (площадной объект / обвалование)

Единицы измерения — метры (в условной системе координат площадки).

5. Тип оборудования
Поле	Тип	Описание
equipment_type	integer	Код типа оборудования

Допустимые значения equipment_type:

Код	Тип оборудования
0	Трубопровод
1	РВС (резервуар вертикальный стальной)
2	Аппарат под давлением
3	Колонна
4	Насос
5	Компрессор
6	Теплообменник
7	Цистерна
8	Трубопровод ПТ
equipment_type = (0, 1, 2, 3, 4, 5, 6, 7, 8)

6. Геометрические характеристики
Поле	Тип	Описание
length_m	number | null	Длина оборудования, м. Актуально для трубопроводов
diameter_mm	number | null	Внешний диаметр, мм. Актуально для трубопроводов
wall_thickness_mm	number | null	Толщина стенки оборудования, мм
volume_m3	number | null	Полный геометрический объём оборудования, м³
7. Заполнение оборудования
Поле	Тип	Описание
fill_fraction	number | null	Степень заполнения оборудования веществом (от 0 до 1)

Интерпретация fill_fraction:

0 — оборудование с газом

1 — оборудование с жидкостью

0 < value < 1 — частичное заполнение (двухфазное состояние)

8. Параметры процесса
Поле	Тип	Описание
pressure_mpa	number	Рабочее давление, МПа
substance_temperature_c	number	Температура вещества, °C
9. Аварийные параметры
Поле	Тип	Описание
spill_coefficient	number	Коэффициент разлива при аварии
spill_area_m2	number	Площадь пролива/разлива, м²

Значения spill_coefficient:

Значение	Условия подстилающей поверхности
5	Неподготовленная грунтовая поверхность
20	Подготовленная грунтовая поверхность
150	Асфальт / бетон

spill_area_m2 трактуется как площадь обвалования или зоны растекания вещества.

10. Временные характеристики аварии
Поле	Тип	Описание
shutdown_time_s	number	Время отключения оборудования, с
evaporation_time_s	number	Характерное время испарения вещества, с
11. Классификация опасного участка
Поле	Тип	Описание
hazard_component	string	Наименование опасного производственного узла или зоны
