Основные поля идентификации
id — уникальный идентификатор вещества (целое число)
​

name — название вещества на русском языке (например, "Бензин", "Толуол", "Метан")
​

kind — код вида опасности вещества (целое число от 0 до 9):
​

0 — Легковоспламеняющаяся жидкость (ЛВЖ)

1 — ЛВЖ (токсичная)

2 — Горючий газ

3 — Горючий газ (токсичный)

4 — Сжиженный горючий газ

5 — Сжиженный горючий газ (токсичный)

6 — Токсичная жидкость (практически неиспаряемая)

7 — Токсичный газ

8 — Сжиженный токсичный газ

9 — Горючая жидкость (практически неиспаряемая)

formula — химическая формула или описание состава (например, "C₇H₈", "Смесь углеводородов C₄–C₁₂")
​

Состав (объект composition)
components — массив компонентов вещества, каждый элемент содержит:
​

name — название компонента

mass_fraction — массовая доля компонента (от 0 до 1)

notes — краткая характеристика вещества (например, "Типовая ЛВЖ", "Горючий газ")
​

Физические свойства (объект physical)
molar_mass_kg_per_mol — молярная масса, кг/моль
​

density_liquid_kg_per_m3 — плотность жидкой фазы, кг/м³ (может быть null для газов)
​

density_gas_kg_per_m3 — плотность газовой фазы, кг/м³ (может быть null для жидкостей)
​

evaporation_heat_J_per_kg — теплота испарения, Дж/кг (может быть null для газов)
​

boiling_point_C — температура кипения, °C
​

Взрывопожарные характеристики (объект explosion)
explosion_hazard_class — класс взрывоопасности (целое число от 0 до 4):
​

0 — невзрывоопасное вещество

1 — самый опасный класс

4 — наименее опасный класс среди взрывоопасных

flash_point_C — температура вспышки, °C (может быть null для газов)
​

lel_percent — нижний концентрационный предел распространения пламени (НКПР), % об. (может быть null для невзрывоопасных веществ)
​

autoignition_temp_C — температура самовоспламенения, °C (может быть null)
​

energy_reserve_factor — коэффициент энергетического запаса (безразмерный):
​

1 — легкий газ (легче воздуха)

2 — тяжелый газ (тяжелее воздуха)

expansion_degree — степень расширения (безразмерный показатель):
​

4 — пары жидкости (меньшее расширение)

7 — газы (большее расширение)

Характеризует способность вещества к объемному расширению при испарении или выбросе

heat_of_combustion_kJ_per_kg — теплота сгорания, кДж/кг (может быть null для негорючих веществ)
​

burning_rate_kg_per_s_m2 — скорость выгорания, кг/(с·м²) (может быть null для газов)
​

Токсичность (объект toxicity)
hazard_class — класс опасности по токсичности (целое число, обычно от 1 до 4):
​

Чем меньше число, тем опаснее вещество

pdk_mg_per_m3 — предельно допустимая концентрация в воздухе рабочей зоны, мг/м³ (может быть null для малотоксичных веществ)
​

lethal_tox_dose_mg_min_per_L — смертельная токсодоза, мг·мин/л (может быть null)
​

threshold_tox_dose_mg_min_per_L — пороговая токсодоза, мг·мин/л (может быть null)
​

Дополнительные характеристики
reactivity — реакционная способность
​

Текстовая оценка: "Стабильно", "Инертен", "Активен", "Окислитель", "Сильный окислитель" и т.п.

odor — характер запаха (например, "Характерный", "Резкий", "Без запаха", "Ароматический")
​

corrosiveness — коррозионная активность
​

Текстовая оценка: "Низкая", "Средняя", "Высокая", "Отсутствует"

precautions — меры предосторожности при обращении (краткая текстовая рекомендация)
​

impact — основные поражающие факторы при аварии (например, "Пожар, взрыв", "Отравление", "BLEVE, факел")
​

protection — средства защиты и локализации (например, "Пена, порошок", "Водяные завесы", "Газоанализ")
​

neutralization_methods — способы нейтрализации/ликвидации разливов (например, "Сорбенты", "Щелочные растворы", "Окисление")
​

first_aid — первая помощь при воздействии (краткие рекомендации, например, "Свежий воздух", "Врач", "Свежий воздух, врач")
​
